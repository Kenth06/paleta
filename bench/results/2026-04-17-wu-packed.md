## Artifact

- Post-opt WASM: 27.8 KB (wasm-opt -O3)
- JS shim: 7.4 KB

## What actually changed

1. **Packed moment layout.** Five cumulative moment tables
   (`wt, mr, mg, mb, m2`) collapsed into one `Vec<f64>` with stride 5.
   `volume()` → `volume5()` returns all 5 moments in a single 8-corner pass
   instead of 5 separate calls. Cuts load count in `variance()` from 40 to
   8 and lets LLVM keep accumulators in SIMD lanes.
2. **Fused `bottom5` / `top5`.** Same fusion for the axis-slab helpers used
   by `maximize()`.
3. **wasm-opt post-processing** with the bulk-memory + saturating-f2i
   feature flags Rust 1.88 emits by default.

## Remaining headroom (not captured yet)

- Manual `v128` intrinsics in the histogram inner loop. Current 17.8k hz
  for 128×128 histogram implies ~1 pixel per 15ns — gather-scatter
  pattern limits autovec.
- JPEG DC-only decoder — the real next win, since decode dominates the
  end-to-end pipeline when palette extraction itself is this fast.
